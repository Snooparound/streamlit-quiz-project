# streamlit-quiz-project

# 음악 기초 상식 퀴즈

학번: 2023204023  
이름: 권기욱

Python과 Streamlit으로 만든 음악 퀴즈 웹 애플리케이션입니다. 회원가입 후 로그인하면 5개의 음악 문제를 풀 수 있고, 각 문제는 20점으로 총 100점 만점입니다.

## 주요 기능

- 첫 화면에 학번과 이름 표시
- 회원가입 및 로그인 기능
- `st.cache_data`를 활용한 퀴즈 데이터 캐싱
- `st.cache_resource`를 활용한 회원 정보 및 풀이 중 답안 임시 저장
- 이전 문제로 돌아가도 입력한 답안 유지
- 정답 채점 및 최종 점수 확인

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run main.py
```

## 데모 영상에서 설명할 내용

1. 앱 첫 화면에서 학번과 이름이 보이는지 확인
2. 회원가입 진행
3. 가입한 계정으로 로그인
4. 캐싱 설명 영역 확인
5. 퀴즈 풀이 중 이전 문제로 돌아가도 답안이 유지되는지 확인
6. 결과 확인 버튼을 눌러 최종 점수 확인

## 참고

회원가입 정보는 실습용으로 앱 실행 중 메모리에 저장됩니다. Streamlit 앱을 완전히 종료하고 다시 실행하면 회원 정보는 초기화됩니다.
